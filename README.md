# imessagedb

Reads the Apple iMessage database

## Installation

```bash
$ pip install imessagedb
```

## Usage

`imessagedb` can be used to parse and view your iMessage messages that are stored 
in your ~/Library/Messages/chat.db file. 

```python
import imessagedb
database = imessagedb.DB()
```
