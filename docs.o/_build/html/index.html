imessagedb¶

Reads the Apple iMessage database

Installation¶

$ pip install imessagedb

Usage¶

imessagedb can be used to parse and view your iMessage messages that are stored in your ~/Library/Messages/chat.db file.

import imessagedb
database = imessagedb.DB()
